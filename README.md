# Smash Walls
cli tool for downloading wallpapers from smashing magazine


## Installation

### Build from source

```
git clone git@github.com:vanadium23/smash-walls.git
cd smash-walls/
go get "golang.org/x/net/html"
go build smash-walls.go
```
